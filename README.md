twallr
======

A simple and nice Twitterwall

requirements
===
- A php-server
- internet connection
- Twitter Application-Token (https://dev.twitter.com/apps)

installation
===
- Copy everything to a dir of your choise, it must be accessable from your webserver.
- Edit the config.php
- Start the server
- Enjoy!
